Directives

 

Directives
Anything that starts with .

 

 

 

 

.data - tells the program to put things in the data section
.text- tells the program to put things in the code section

 

 

.word -> 4 bytes (int!?)

 

.space bytesCount - padding

 

.asciiz "string"

-> puts in s t r i n g 0

(null terminator)

 

.align n

2^n

CPU reads 4 bytes every time

Can't read non-aligned memory address
(ie can read 0-3, can't read 1-4)


ie .align 2; 2^2 - 4

 

.byte -> 1 byte

 

 

 

Can supply multiple
.byte 1,2,3,4 --> 1 2 3 4

.word 1,2,3,4 --> 1 0 0 0 2 0 0 0 3 0 0 0 4 0 0 0
