RETURN VALUE

       On success, the number of bytes written is returned (zero

       indicates   nothing   was  written).   On  error,  -1  is

       returned, and errno is set appropriately.

 

       If count is zero and fd refers to a  regular  file,  then

       write()  may return a failure status if one of the errors

       below is detected.  If no errors are detected, 0 will  be

       returned  without  causing any other effect.  If count is

       zero and fd refers to a file other than a  regular  file,

       the results are not specified.
